🤖 AI FRAUD DETECTION FEATURES:

The system automatically analyzes EVERY transaction with:

✓ Factor 1: Amount Analysis
   - Checks if amount > ₹50,000
   - Risk: +30 points

✓ Factor 2: Rapid Transactions
   - Detects >3 transactions in 5 minutes
   - Risk: +25 points

✓ Factor 3: New Account Check
   - Flags new accounts (<24hrs) sending >₹10,000
   - Risk: +35 points

✓ Factor 4: Velocity Check
   - Monitors total sent in last hour
   - If >₹1,00,000 → Risk: +40 points

✓ Factor 5: Round Number Detection
   - Flags amounts like ₹50,000, ₹1,00,000
   - Risk: +15 points

✓ Factor 6: Multiple Receivers
   - Tracks different receivers in 24h
   - If >5 receivers → Risk: +20 points

✓ Factor 7: Time Analysis
   - Flags transactions 12 AM - 5 AM
   - Risk: +15 points

✓ Factor 8: Keyword Detection
   - Scans for: urgent, emergency, lottery, prize
   - Risk: +10 points per keyword

📊 RISK SCORING SYSTEM:

Risk Score Calculation:
• All risk factors are added up
• Score ranges from 0% to 100%

Decision Rules:
• 0-39%   → ✅ LOW RISK    - Transaction approved
• 40-59%  → ⚠️  MEDIUM RISK - Transaction flagged
• 60-100% → 🚫 HIGH RISK   - Transaction BLOCKED
